\# NitroSense AI



AI-powered web application for real-time nitrogen estimation in wheat crops.



\## Features



\- Upload wheat leaf images

\- Enter fertilizer amount (0-210 kg/ha)

\- Enter air temperature

\- Select sowing and capture dates

\- Get instant nitrogen content predictions



\## Model Performance



\- \*\*Model\*\*: DenseNet121 with Feature Pyramid Network (FPN)

\- \*\*R² Score\*\*: 0.7897

\- \*\*RMSE\*\*: 0.3044

\- \*\*MAE\*\*: 0.2398
