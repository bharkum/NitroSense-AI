## ✨ Features

- 📸 Upload or capture wheat leaf images
- 🌡️ Enter agronomic data (fertilizer amount, sowing date, capture date, air temperature)
- 🧠 Get instant nitrogen content predictions using deep learning
- 📊 Receive actionable recommendations for fertilizer management
- 🖼️ Real-time image processing with background removal and edge detection
