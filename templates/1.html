How NitroSense AI Works

Multimodal Intermediate Fusion Architecture Flow

📥 Input Layer

Image Input
  • Wheat leaf image
  • Processed through CNN Architecture
  • Input size: 224x224x3 or 299x299x3
Tabular Input
  • Nitrogen Fertilizer Level (kg/ha)
  • Days After Sowing

🖼️ Image Processing Branch

CNN Architecture
  • Pre-trained on ImageNet
  • Transfer Learning
  • Extracts hierarchical features

📊 Tabular Data Processing Branch

MLP Layer Processing
  • Extracts tabular features from agronomic data
  • Processes Nitrogen Level and Days
  • Feature encoding and normalization

🔄 Intermediate Fusion Layer

Image Features + Tabular Features → Combined Vector

✨ This fusion allows the model to learn cross-modal relationships

🔄 The network correlates visual symptoms with agronomic conditions

🎯 Prediction Network

Dense Layers Processing
  • Fused features processed through dense layers
  • Final output layer with linear activation
  • Output: Normalized Nitrogen Content

CNN Model Architecture Summary

CNN Models: ResNet50, Xception, DenseNet121, MobileNetV3, EfficientNetV2

Loss Function: Mean Squared Error (MSE)

Evaluation Metrics: RMSE, MAE, R²

Input Image Size: 224x224x3 or 299×299×3

Output: Normalized Nitrogen Content (2-5% range)

Trainable Parameters: Varies by model

Why Multimodal Fusion?
By combining visual symptoms with agronomic context, the model achieves higher accuracy than either modality alone. For example, yellowing leaves might indicate deficiency, but the model also considers growth stage and fertilizer applied to make a more informed prediction.
